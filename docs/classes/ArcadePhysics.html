ArcadePhysics Class

Defined in: src\components\ArcadePhysics.ts:10
Module: Components
Parent Module: Kiwi

Ported from Flixel, most functions operation identically to the original flixel functions, though some have been split into multiple functions. Generally where functions originally accepted either groups or gameobjects within the same argument, the ported functions one or the other. http://www.flixel.org/ http://www.adamatomic.com/
